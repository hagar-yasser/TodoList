# TodoList
## How to run the task
* Write the next commands in terminal 
  ```
  git clone https://github.com/hagar-yasser/TodoList.git

  ```
* Open Workbanch then 
  * Connect to server localhost:3306
  * Type this query
     ```
     
      USE TodoList;
     
     ```
  
* Open TodoList folder in Intellij Idea
* Run the Main Class
